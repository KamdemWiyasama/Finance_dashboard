Finance Dashboard – Full-Stack Web Application

Overview
Finance Dashboard is a full-stack web application that allows users to track expenses, store financial data persistently, and convert currencies using real-time exchange rates from an external REST API.
This project demonstrates backend development, API integration, database management, and front-end design using Python and Flask.

Features

Add and manage categorized expenses
Persistent storage using SQLite
Real-time currency conversion via external REST API
Dynamic data rendering using Flask templates
Input validation and structured error handling
Clean and responsive user interface

Tech Stack

Backend: Python, Flask, SQLite, Requests (API calls), 

Frontend: HTML, CSS, JavaScript

Other Tools: Git, VS Code

API Integration:
The application integrates with a public exchange rate API to retrieve real-time currency conversion rates. JSON responses are parsed and processed to dynamically convert expense values.
